Program - vigenere.cpp

Code
vigenere.cpp contains the program for encrypting the text present in text.txt file.

Input
text.txt contains the text to be encrypted.
First line contains the key which is used to encrypt the text.
The rest of the file is the text to be encrypted.

Run the Program
1. Compile the code ->> g++ vigenere.cpp
2. Execute the code giving the file as input ->> ./a.out < test.txt

Output

3. Key found by cryptanalysis
