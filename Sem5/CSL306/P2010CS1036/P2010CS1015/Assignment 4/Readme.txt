__________________________RUNNING THE SAMPLE________________________________

In the terminal,simply write the command 'make'
In order to choose the GUI environment with which you want to work, the user of our Common API needs to import one of the following files:

	from AnyGUI.tkGUI import *      #for PyTkinter
	from AnyGUI.qtGUI import *	#for PyQt
	from AnyGUI.gtkGUI import *	#for PyGtk
	from AnyGUI.wxGUI import *	#for wxPython
